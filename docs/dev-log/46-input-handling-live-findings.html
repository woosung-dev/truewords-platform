chowon.in

4.1 모바일 앱 "질문하기" 화면

placeholder 두 줄 가이드 ("고민이나 질문을 입력해주세요 / 내용이 구체적일수록 답변이 정확해요") + 맞춤 설정 3종(교단 / 답변 모드 / 공개 여부) + 정교한 면책 4문장 ("ChatGPT와 달리 존재하지 않는 성경 구절을 인위적으로 생성하지 않으며... 목회자 검수위원회 신학적 검증 완료... 이단 데이터 학습 신학적 방화벽 구축")

4.2 답변 모드 페르소나 5종 ★★ (가장 큰 발견)

표준(추천) / 신학과 교수 / 목회 상담자 / 초신자 양육자 / 유년부 선생님 — 동일 질문에 5가지 페르소나로 답변. "목회 상담자" 페르소나는 위로/상담 중심 = TrueWords 위급 라우팅(ADR-45 P0-1) 자연 구현 경로

4.3 교단 설정 4종+

전체 교단 / 장로교 / 감리교 / 침례교 — 선택 교단의 신학적 특징이 답변에 반영. TrueWords는 "신학 강조점 토글" (원리 중심 / 가정 중심 / 실천 중심 등)으로 변형 가능

4.4 답변 페이지 본체 (모바일 앱 #4 — 인용 카드 3-탭 클릭 결과 시리즈의 default)

각 인용 카드마다 독립된 3-탭(💡 해설 / 📖 본문 / ✏️ 노트) — 답변 단위가 아니라 각 성경 구절 카드 단위 토글. 첫 인용 = 욥기 1:6-7(헤더), 본문 안 인용 3개 = 욥기 6:4 / 히브리서 2:4 / 베드로전서 5:8 → 총 4개 인용 카드.

답변 본문 6단 정정: ① 도입 1문단(굵은 키워드 강조) → ② 핵심적으로 보면(3 글머리) → ③ 성경이 보여주는 모습(인용 카드 3개) → ④ 기독교적으로 중요한 균형(3 글머리) → ⑤ 실천적으로는(4 글머리) → ⑥ 마지막 한 단락 + 상황 맞춤 기도문(별도 박스, "예수 그리스도의 이름으로 기도드립니다. 아멘.").

하단 floating action bar(↩ 새 질문하기 / 🔖 북마크 / 🔗 공유) — 모바일 앱에서만 보임. Playwright 공유 페이지 실측에서는 미노출 (앱 전용으로 추정).

4.5 인용 카드 💡 해설 탭 클릭 결과 (모바일 앱 #5)

인용 카드 안에서 💡 해설이 활성화되면: "욥기 1:6-7 해설" 헤더 + 해설 본문(3 단락) + 끝에 일러스트 카드 ("욥기 1장 전체 읽기 / 욥의 믿음과 시험").

일러스트 카드 클릭 시 별도 본문 페이지로 jump → §4.6 화면.

4.6 일러스트 카드 클릭 시 이동한 본문 페이지의 AI 해설요약 탭 (모바일 앱 #6)

§4.5의 일러스트 카드를 따라 도착한 별도 성경 본문 페이지:
• 헤더 ← 욥기 1장 ▼ + 번역본 토글 [개역개정] + 헤더 우측 🔍 AI(본문 컨텍스트로 AI에 직접 질문)

• 글로벌 nav: 홈 / 통독표 / 노트 / 북마크 / ⚙

4.7 답변 끝 추천 follow-up = 5개 (3·4·5번 블러) + 카카오 로그인 유도 (정정)

4.8 공유 페이지 freemium paywall (Playwright 신발견 ★)
